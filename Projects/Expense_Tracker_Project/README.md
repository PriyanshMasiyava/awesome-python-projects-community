# 📖 Overview

Expense Tracker Pro is a modern desktop application developed using **Python** and **Tkinter** that helps users efficiently manage their daily expenses.

The application provides a clean graphical interface for recording expenses, tracking budgets, exporting reports, and visualizing spending through interactive charts.

It is designed as a complete beginner-to-intermediate Python project demonstrating GUI development, database integration, data visualization, and report generation.

# 🚀 Installation

## Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/Expense_Tracker_Pro.git
```

Move into the project

```bash
cd Expense_Tracker_Pro
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run

```bash
python main.py
```
