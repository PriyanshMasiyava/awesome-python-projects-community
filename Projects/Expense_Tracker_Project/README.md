# 💰 Expense Tracker Pro

A professional desktop Expense Tracker built with Python.

## Features

- Tkinter GUI
- SQLite Database
- Expense Analytics
- Charts
- CSV Export
- PDF Reports
- Calendar Picker
- Dark Theme
- Advanced Search

## Project Structure

```
Expense_Tracker/

main.py
app.py
config.py
theme.py
database/
exports/
assets/
screenshots/
charts/
```

## Installation

```bash
pip install -r requirements.txt
```

Run

```bash
python main.py
```

## Technologies

- Python
- Tkinter
- SQLite
- Matplotlib
- ReportLab
- Pandas
- tkcalendar
